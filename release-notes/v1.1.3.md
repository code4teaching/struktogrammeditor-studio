# VisuStruct 1.1.3

Release **after** **1.1.2** (GitHub `master`).

## Tabs

- Close diagram tabs via **middle-click**, the tab **context menu**, and **Ctrl+W** / **Cmd+W**, with sensible behaviour for the last tab and unsaved changes.
- Clicking the **tab header** reliably switches the active diagram; the close control is a **label** instead of a **JButton** so it no longer intercepts tab-selection clicks.

## Left palette (trash)

- **Left-click** on the trash icon deletes the **currently selected** (yellow-highlighted) block, with the same **confirmation dialog** as **Delete** in the context menu.
- **Drag** a block from the diagram onto the trash (unchanged): removes it from the diagram.

## Code generator

- Extensions for **Java scope** and **`CodeGenRules`**, including **`StruktogrammElementListe`** and **`CodeErzeuger`**.

## Build

- Maven: **`de.visustruct:visustruct:1.1.3`**
- Fat JAR: **`visustruct-1.1.3.jar`** (`./mvnw package` → `target/`)
