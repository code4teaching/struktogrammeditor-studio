# VisuStruct 1.2.1

Patch release **after** **1.2.0**.

## Simulation

- **Verzweigung:** Zweige mit Fallnamen `true` / `false` werden nach Namen gewählt (nicht nur nach XML-Reihenfolge) — z. B. nach „Zweige vertauschen“ und Speichern.
- **Reset (Stopp):** Schrittliste wird wieder aus dem Diagramm aufgebaut; **neue Eingabe** bei erneutem Lauf (nicht mehr die erste Zahl).
- **Modulo / Vergleich:** numerische `String`-Werte in Ausdrücken robuster.

## Speichern

- **Element-Editor:** ausstehender Text wird vor **Speichern** ins Diagramm übernommen (wie vor Simulation).
- **macOS:** Speicherdialog beim ersten Speichern zuverlässiger (`invokeLater`); fehlende Zielordner werden angelegt.

## Build

- Maven: **`de.visustruct:visustruct:1.2.1`**
- Fat JAR: **`visustruct-1.2.1.jar`** (`./mvnw package` → `target/`)
